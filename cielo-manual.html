CIELO Backgrounds User Manual

Dynamic Matte Backgrounds for Blender

01 Installation

1.1 Setting up the Asset Library

  1. Unpack the provided zip file somewhere in your system.
  2. Go into the Blender Preferences and open the File Paths section.
  3. Click on the + icon to add an asset library.

  4. Navigate into your extracted CIELO Backgrounds folder and click Add Asset Library.
Important Path Warning:

Do not click on the Sources directory. The path should end in \CIELO Backgrounds\ and not in \Sources\.

Finally, ensure your new asset library is correctly named:

1.2 Allowing Python Execution

It is recommended to allow automatic Python execution so the asset UI loads automatically when dropping an asset into your scene. This setting is found under Save & Load preferences.

02 Usage

  1. Open an Asset Browser window.
  2. Switch to the CIELO Backgrounds asset library to see the full list of matte backgrounds.

  3. Drag and drop one of the CIELO backgrounds into your scene.
  4. Go to the World Properties tab. Look for the CIELO panel to access the control interface.

For a complete tour of all available settings go to the main page.
