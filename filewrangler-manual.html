File Wrangler User Manual

Batch Processing Automation for Blender

01 Introduction

Welcome to File Wrangler! This Blender addon is a powerful batch processing tool designed to automate repetitive tasks across multiple .blend files. Whether you need to clean up entire project directories, change file settings, or run custom scripts, File Wrangler saves you hours of manual work by handling it all in one go.

The core philosophy is to set up a series of actions, apply filters to target specific files, and execute the process safely and efficiently.

02 Installation

  1. Download the file_wrangler.zip file.
  2. In Blender, go to Edit > Preferences > Add-ons.
  3. Click the Install... button at the top-right.
  4. Select the downloaded .zip file.
  5. Enable the addon by ticking the checkbox next to its name.

03 The Interface

Open the Sidebar in the 3D Viewport by pressing N and look for the File Wrangler tab.

The panel is organized into three main sections: Setup & I/O, Filename Filters, and Actions.

04 Configuration

4.1 Setup & I/O

Defines what files to process and where to save them.

  • Source Directory: The main folder containing your .blend files.
  • Output Directory (Recommended):
    • If set: Copies structure and saves modified files here. Safe & Non-destructive.
    • If blank: Overwrites original files. Use with caution.
  • Search Recursively: If enabled, searches subfolders.
  • Write Log File: Creates file_wrangler_log.txt recording all actions and errors.

4.2 Filename Filters

Precisely target files (e.g., characters, environments).

  • Include files containing: Process only files matching these keywords (e.g., character, prop).
  • Exclude files containing: Skip files matching these keywords (e.g., test, old, broken).

05 Actions

Choose operations to perform on the targeted files.

Delete Backup Files (.blend1)

Scans the directory and deletes all backup files to clean up clutter. Runs independently of the main loop.

Purge Unused Data

Performs a recursive "Orphans Purge" on each file, removing unused materials, textures, and node groups.

Find Missing Files

Attempts to locate and relink missing external assets. Specify a Search Path or default to the Source Directory.

Run Custom Script

Run a Python script on every file. Use bpy and os modules.

  • External File: Load a .py file from disk.
  • Internal Text: Use a script from the Text Editor.
File Paths & Compression
  • Paths: Set to Relative (portable) or Absolute (fixed structure).
  • Compression: Enable to save disk space, or disable for faster opening.

06 Execution & Safety

Running the Process

  • Dry Run: A safe preview. Prints a list of affected files to the System Console/Log without making changes. Always use this first.
  • Start Batch Process: Begins the operation. A progress bar allows you to Cancel if needed.
⚠️ Important Warnings
  • ALWAYS BACK UP YOUR DATA before running a batch process.
  • Using a separate Output Directory is the recommended workflow.
  • Actions like "Delete Backup Files" are permanent.

07 Preferences

Customize the sidebar tab name:

  1. Go to Edit > Preferences > Add-ons.
  2. Expand "File Wrangler".
  3. Change the UI Tab Name field.
