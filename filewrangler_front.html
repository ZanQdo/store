File Wrangler

Tired of repetitive tasks? Automate cleaning, managing, and modifying your .blend files. Work smarter, not harder.

From tedious to automated.

Real World Solutions

Solve complex pipeline problems instantly.

01. The Project Cleanup

The Solution: One click deletes backups, purges unused data-blocks, and compresses files to save disk space. A task that takes hours is done in seconds.

02. The Migration Nightmare

The Solution: Don't open files manually! File Wrangler automatically scans, finds missing files, and relinks everything for the entire project.

03. The Pipeline Power-Move

The Solution: Write a simple Python script, point File Wrangler to it, and watch it apply the changes to every file automatically.

Safe and Powerful

Dry Run Mode
Worried about making changes? Use Dry Run to see a list of all files that will be affected before you commit.
Safety Sandbox
For ultimate safety, set an Output Directory. File Wrangler will save modified versions there, leaving your original files untouched.
Simple Interface
