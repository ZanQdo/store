Snapshots User Manual

Professional Version Control for Blender

01 Introduction

Welcome to Snapshots, a professional version-control workflow for Blender. Snapshots is designed to liberate your creative process by removing the fear of commitment. By creating non-linear "checkpoints" in your work, you can explore radical creative tangents, A/B test lighting setups, and prototype complex procedural systems, all while knowing your primary vision is safe and restorable with surgical precision.

02 Installation

  1. Download the snapshots.zip file.
  2. In Blender, go to Edit > Preferences > Add-ons.
  3. Click the Install... button at the top-right.
  4. Select the downloaded .zip file.
  5. Enable the addon by ticking the checkbox next to Snapshots.

03 Configuration & Pipeline

Access these settings in Edit > Preferences > Add-ons > Snapshots.

Global Snapshots Directory

By default, Snapshots are saved relative to your project. Set a global directory to centralize backups to a secondary drive or network server, which is ideal for studio environments.

UI Tab Customization

Change the UI Tab Name to move the Snapshots panel to a specific location in your N-panel sidebar (e.g., "Pipeline" or "Exploration").

04 Capturing & Managing History

Press N in the 3D Viewport and select the Snapshots tab to view your timeline.

Creating a Snapshot

Click Create Snapshot. If your file hasn't been saved yet, it will read Create Snapshot & Save to initialize the system.

Interactive Thumbnail Crop

When creating a snapshot, check Capture Thumbnail. After clicking OK, you will enter an interactive mode in the 3D Viewport. Simply click and drag to draw a square region to establish a precise, cropped visual reference for that specific save state.

Managing Your Timeline

  • Annotations (Notes): Click any snapshot to expand it and type detailed notes to track your process.
  • Actions (Cog Icon): Rename descriptions or permanently delete snapshots to keep your list clean.
  • View Full Thumbnail: Under the thumbnail image, click the View Full Thumbnail button to pop open a high-resolution version of your capture in Blender's standalone image player.

05 Contextual Editor Integration

The Snapshots panel dynamically adapts to your workspace, allowing you to create partial snapshots of specific data blocks without saving the entire file.

Node Editor

When working in the Shader, Geometry Nodes, or Compositor editors, the sidebar adapts to target the active Node Tree. Quickly version-control complex node setups independently of the scene.

Dopesheet & Graph Editor

In animation editors, the add-on recognizes the active Action data-block. You can snapshot different animation takes, A/B test timing, and instantly restore specific performances.

06 Surgical Restoration

Snapshots allows you to mix and match different versions of your project. You don't always have to revert the whole file; you can pull just the ideas you need.

6.1 Property-Level Recovery (Right-Click)

This is the most powerful way to use Snapshots.

  1. Right-click any field (Roughness, Focal Length, Location, etc.).
  2. Select Restore Property from Snapshot.
  3. Pick a snapshot from the list. Only that specific value (and any attached drivers or keyframes) will revert.

6.2 Component & Data-Block Restoration

  • Materials & Node Trees: Right-click the material name or node group to restore the entire node tree configuration.
  • Modifiers & Constraints: Right-click the name of a modifier or constraint in the stack to revert its entire configuration.
  • Actions: Restore specific animation takes by right-clicking the Action name.

6.3 Bulk Restoration (Viewport Menu)

Select multiple objects in the 3D Viewport and Right-click > Restore from Snapshot.

  • Restore Data: Reverts underlying meshes/curves/lights while keeping current positions.
  • Restore Object Properties: Reverts transforms (Location, Rotation, Scale).
  • Restore Modifiers & Constraints: Synchronizes entire modifier stacks across multiple objects.
  • Restore Materials: Reverts all material slot assignments for the selection.

07 Recovery & Scene Tools

7.1 Recover Deleted Data

If you set aside an idea earlier and deleted it from your scene, you can bring it back:

  1. Expand a global snapshot in the sidebar.
  2. Click Recover Deleted Data.
  3. The system automatically identifies every object or data-block present in the snapshot but missing from your current scene.
  4. Select the items and click OK. Objects will be restored into their original collections if available.

7.2 Scene-Level Restoration

Inside the expanded snapshot panel, use the quick-action buttons to restore high-level environments:

Scene: Settings & Render Props
World: Lighting & Shaders
View Layer: Render Passes

08 Technical Compatibility

Geometry: Meshes, Curves, Lattices, Text, Armatures, and Bones.
Animation: Action Datablocks, Drivers, and Keyframed properties.
Nodes: Geometry Nodes, Shader trees, and the Compositor.
Metadata & Misc: Grease Pencil, Palettes, and Cameras.
