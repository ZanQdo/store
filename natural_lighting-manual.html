02 Installation

2.1 Setting up the Asset Library

  1. Unpack the provided zip file.
  2. Go to Preferences > File Paths.
  3. Click the + icon to add a library.
  4. Navigate to the Natural Lighting folder and click Add Asset Library.
Important Path Warning:

Do not click on the Sources directory. The path should end in \Natural Lighting\.

Make sure your new asset library is correctly named:

2.2 Allowing Python Execution

Enable automatic python execution in Save & Load preferences so the UI loads automatically.

03 Usage & Interface

  1. Open the Asset Browser and select Natural Lighting.

  2. Drag an asset into your scene and position the lamp.
  3. Select the light and go to the Light Data Properties (Green Bulb Icon). Look for the Natural Lighting tab.

04 Light & Bloom Settings

Main Light

  • Power & Exposure: Controls energy in Watts and exponential scaling.
  • Spot Size & Softness: Controls the angle and edge softness.
  • Temperature (Kelvin): Based on black body radiation.
    2500K = Warm Yellow | 6500K = Daylight White | 12000K = Cool Blue
  • Tint: Colorize the light with an RGB picker.

Bloom (Warm Edge)

Adds a soft warm glow, mostly visible at shadow edges.

  • Strength: Factor of main light power (0.5 = half power).
  • Temperature: Usually half the main temp for a "warm" effect.
  • Softness: Usually much softer than the main light for a glowy look.

05 Projection & Animation

5.1 Projection (Cast Shadows)

Controls the cast shadow pattern effect.

Strength:
0.0 turns effect off. Lower than 1.0 is subtle.
Softness:
Higher values blur the shadows.
Transform:
Rotate, Scale, and Offset the pattern.
Color Grading:
Hue, Saturation, Brightness, Contrast.

5.2 Clip Settings

Natural Lighting is based on curated movie clips.

  • Length: Total available length of the clip in frames.
  • Frame Offset: Offset the clip to select a different starting point.
